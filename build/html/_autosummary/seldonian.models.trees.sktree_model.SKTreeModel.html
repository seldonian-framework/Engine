Methods

backward_pass(ans, theta, X)¶

Return the Jacobian d(forward_pass)_i/dtheta_{j+1}, where i run over datapoints and j run over model parameters.

Parameters:

  • ans – The result of the forward pass function evaluated on theta and X

  • theta – The weight vector, which isn’t used in this method

  • X – The features

predict(theta, X, **kwargs)¶

Do a forward pass through the sklearn model. Must convert back to numpy array before returning

Parameters:

  • theta (numpy ndarray) – model weights (not probabilities)

  • X (numpy ndarray) – model features

Return pred_numpy:

model predictions

Rtype pred_numpy:

numpy ndarray same shape as labels
