seldonian.hyperparam_search.HyperparamSearch¶

class HyperparamSearch(spec, all_frac_data_in_candidate_selection)¶

Bases: object

__init__(spec, all_frac_data_in_candidate_selection)¶

Object for finding the best hyperparameters to use to optimize for probability of returning a safe solution for Seldonian algorithms. Note: currently only used to find optimal data split.

List of hyperparameters to optimize: - Percentage of data in candidate and safety datasets

Parameters:

  • spec (Spec object) – The specification object with the complete set of parameters for running the Seldonian algorithm

  • all_frac_data_in_candidate_selection (numpy.ndarray) – Array containing the values of fraction of data in candidate solution that are being considered
