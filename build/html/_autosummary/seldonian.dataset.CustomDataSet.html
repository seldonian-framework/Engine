seldonian.dataset.CustomDataSet¶

class CustomDataSet(data, sensitive_attrs, num_datapoints, meta, **kwargs)¶

Bases: DataSet

__init__(data, sensitive_attrs, num_datapoints, meta, **kwargs)¶

A flexible container for holding data of arbitrary form for the custom regime.

Parameters:

  • data (numpy.ndarray or list) – A list of data points, where each data point can be an object of any form.

  • sensitive_attrs (numpy.ndarray, defaults to []) – Sensitive attribute array for each data point

  • num_datapoints (int) – Number of data points

  • meta (RLMetaData) – Metadata object
